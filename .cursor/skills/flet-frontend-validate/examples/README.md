# Exemplos auxiliares (Flet)

Arquivos **não executáveis como app completo** — servem de referência rápida para padrões compatíveis com o SDK usado no projeto (tipicamente Flet 0.21.x).

| Arquivo | Objetivo |
|---------|----------|
| `layout_row_wrap.py` | Substituir `Wrap` por `Row` com `wrap=True` |
| `clickable_surface.py` | Área clicável sem `InkWell` |
| `circle_avatar_url.py` | Avatar com imagem remota |

Consulte o comentário no topo de cada `.py`.
