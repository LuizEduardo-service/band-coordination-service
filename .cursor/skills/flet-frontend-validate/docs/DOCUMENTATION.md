# Documentação e referências Flet

Use estes links ao validar ou corrigir o frontend. A documentação do site pode descrever APIs mais recentes que o `flet` fixado no projeto; sempre cruzar com `flet.__version__` local.

## Oficial

| Recurso | URL |
|--------|-----|
| Site do projeto | https://flet.dev/ |
| Documentação (portal principal) | https://flet.dev/docs/ |
| Documentação espelho / intro | https://docs.flet.dev/ |
| Visão geral da API (controles, serviços, CLI) | https://docs.flet.dev/api-reference/ |
| Repositório (issues, releases, exemplos) | https://github.com/flet-dev/flet |
| Pacotes PyPI | https://pypi.org/project/flet/ |
